Introduction

In this example, we create 9 squares of various sizes and positions, and using a uniform variable projection matrix in the shader keep the objects looking like squares when the screen is resized.

The graphics window will display 9 multicolored squares. The title bar of the window will also display the number of frames per second for the last second.

User Options

  • Escape: Ends the program.
  • M: Toggles between fill mode and line mode to draw the triangles.
  • R: Resets the window size to 700 X 500.
  • A: Resets the window size to 600 X 600.
  • F1: Sets vertex 0 of the box to (1, 0.3, 0.21)
  • F2: Sets vertex 0 of the box to (1, 0.3, 0.21)
  • F5: Resets the box center to (0, 0).
  • F10: Saves a screen shot of the graphics window to a png file.
  • 1 - 9: Sets the current box to be altered.
  • Left: Moves the box to the left.
  • Right: Moves the box to the right.
  • Up: Moves the box up.
  • Down: Moves the box down.
Note
Note that the shader programs "AspectRatioVert.glsl" and "PassThroughFrag.glsl" are expected to be in the same folder as the executable. Your graphics card must also be able to support OpenGL version 3.3 to run this program.
